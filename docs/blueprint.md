# **App Name**: TMSL Static Site

## Core Features:

- Homepage: Display featured college content in a carousel format.
- Informational Pages: Static pages for departments, clubs, facilities, and blog content.
- About Team: Dedicated page with details on the 5-member development team.
- Contact Form: Simple form to collect messages from visitors.
- AWS S3 Hosting: Host static files on AWS S3 for high availability and low cost.

## Style Guidelines:

- Primary color: A vibrant blue (#2E9AFE) to convey trustworthiness, knowledge, and authority.
- Background color: A light, desaturated blue (#EBF5FB), nearly white.
- Accent color: An analogous teal (#2E9AFC), less saturated and darker than the primary.
- Body font: 'PT Sans', a humanist sans-serif font, providing a modern yet warm feel for general text.
- Headline font: 'Space Grotesk', a bold sans-serif font, for impactful headlines and titles.
- Use a set of consistent, outline-style icons for navigation and to represent departments/clubs.
- Grid-based layout for consistent spacing and alignment across all pages.
- Subtle page transitions and loading animations to enhance user experience.
